## plus&touhisuuretu

- plusは標準入力を用いて数値を1行ずつ読み取りそれらすべて足し合わせた合計を出力するプログラムです。
- touhisuuretuは等比数列の指定した初項(a1)、公比(r)、項数(n)を計算するプログラムです。
